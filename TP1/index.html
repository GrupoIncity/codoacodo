Bs As - Octubre

Bootstrap es el marco de interfaz de usuario de código abierto más popular. Fue desarrollado por Mark Otto, Jacob Thornton el 19 de agosto de 2011. Bootstrap es una colección de clases CSS y funciones de JavaScript y se utiliza para el diseño receptivo y la construcción de sitios y aplicaciones receptivos, primero para dispositivos móviles. Por lo general, funciona en un sistema de cuadrícula para crear diseños de página con la ayuda de filas y columnas y es compatible con todos los navegadores para crear sitios web receptivos.
